# :rocket: How to run the app locally

### 1. Clone this repo

### 2. Install dependencies

`yarn install`

### 3. Start and view the app :eyes:

`yarn start` :horse_racing:

Then, in your browser, open http://localhost:8000/ to view it! :tada: :tada:
